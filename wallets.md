# Supported Wallets

MegaBridge supports a wide variety of external wallets for connecting and bridging.

## Recommended Wallets

### MetaMask
The most popular browser extension wallet. Available for Chrome, Firefox, Brave, and Edge.
- [Download MetaMask](https://metamask.io/)

### OKX Wallet
Feature-rich wallet with multi-chain support.
- [Download OKX Wallet](https://www.okx.com/web3)

### Rabby Wallet
Security-focused wallet with transaction simulation.
- [Download Rabby](https://rabby.io/)

## Other Supported Wallets

- **Coinbase Wallet** - Mobile and browser extension
- **Rainbow** - Beautiful mobile wallet
- **Phantom** - Multi-chain wallet
- **Zerion** - DeFi-focused wallet
- **Bybit Wallet** - Exchange-integrated wallet
- **Bitget Wallet** - Multi-chain support
- **WalletConnect** - Connect any WalletConnect-compatible wallet

## Connecting Your Wallet

1. Click **Connect Wallet** on MegaBridge
2. Select your wallet from the list
3. If using a browser extension, approve the connection popup
4. If using WalletConnect, scan the QR code with your mobile wallet

## Network Configuration
