# Networks

MegaBridge connects two networks: Base and MegaETH.

## Base

Base is a secure, low-cost, builder-friendly Ethereum L2 built to bring the next billion users onchain.

### Network Details
| Property | Value |
|----------|-------|
| Chain ID | 8453 |
| Currency | ETH |
| RPC URL | https://mainnet.base.org |
| Block Explorer | https://basescan.org |

### Adding Base to Your Wallet

Most wallets will automatically prompt you to add Base when needed. If you need to add it manually:

1. Open your wallet settings
2. Add a new network
3. Enter the details above

## MegaETH

MegaETH is a high-performance Ethereum Layer 2 network designed for speed and efficiency.

### Network Details
| Property | Value |
|----------|-------|
| Chain ID | 4326 |
| Currency | ETH |
| RPC URL | https://rpc-secret-mega.poptyedev.com/ |
| Block Explorer | https://mega-explorer-leaked.poptyedev.com/ |

### Adding MegaETH to Your Wallet

MegaBridge will automatically prompt you to add MegaETH when bridging. To add manually:

1. Open your wallet settings
2. Add a new network
3. Enter the details above

## Bridging Between Networks

When you initiate a bridge transaction, MegaBridge will:
1. Detect your current network
2. Prompt you to switch to the source network if needed
3. Guide you through the transaction process
